## C 0/10
De while loop is niet helemaal compleet.  De gebruiker moet de volgende input kunnen geven:

“a N” Geeft aan dat passagier met ID N achteraan in de rij gaat staan.
“v N” Geeft aan dat passagier met ID N vooraan in de rij gaat staan.
“A” De persoon achteraan verlaat de wachtrij.
“V” De persoon vooraan verlaat de wachtrij.
“P” Print de huidige rij
“S” Stop het programma
“? N” Print de plaats in de rij van de eerste passagier met ID N.
“# i” Print de ID van de passagier die op plaats i staat in de rij.

Je mag ervan uitgaan dat de gebruiker alleen geldige input levert.

Voorbeeld output:

Welke actie?:a 3

Welke actie?:a 6

Welke actie?:A

Welke actie?:a 5

Welke actie?:P

3, 6, 5,

Welke actie?:S
